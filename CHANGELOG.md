# Changelog
Todos os registros de mudanças relevantes neste projeto serão documentados neste arquivo.

O formato é baseado em Keep a Changelog (https://keepachangelog.com/pt-BR/1.0.0/)
e este projeto adota versionamento semântico (https://semver.org/lang/pt-BR/).

---

## [1.0.0] - 2025-02-08
### Adicionado
- Dashboard principal com resumo total das assinaturas Xbox Games
- Indicadores de faturamento total por serviço
  - Xbox Games Games
  - EA Play Season Pass
  - Minecraft Play Season Pass
- Segmentação por Tipo de Assinatura:
  - Annual
  - Monthly
  - Quarterly
- Segmentação por Tipo de Plano:
  - Core
  - Standard
  - Ultimate
- Gráfico de distribuição por Auto Renewal (Sim / Não)
- Gráfico de faturamento por Tipo de Plano
- Layout visual customizado para análise gerencial
- Atualização automática dos gráficos com base nos filtros
- Aba de Ajuda interna no Excel
- Manual do Usuário
