Manual do Usuário — Dashboard Xbox One

Visão Geral



Este Dashboard foi desenvolvido para facilitar a visualização e análise das vendas de assinaturas do Xbox One. Através de gráficos e indicadores visuais, é possível acompanhar rapidamente o desempenho dos principais serviços e planos.



O painel foi projetado para ser simples, intuitivo e interativo.



Como Utilizar os Filtros



O Dashboard possui dois filtros principais:



1\. Tipo de Assinatura



Permite filtrar os dados pelo período da assinatura:



Annual (Anual)



Monthly (Mensal)



Quarterly (Trimestral)



Utilize este filtro para analisar o impacto do tipo de contrato nos resultados.



2\. Tipo de Plano



Permite filtrar os dados pelo plano contratado:



Core



Standard



Ultimate



Utilize este filtro para identificar quais planos geram maior faturamento.



Indicadores Principais



O painel apresenta os seguintes indicadores:



Total de Assinaturas Xbox



Total de Assinaturas EA Play Season Pass



Total de Assinaturas Minecraft Play Season Pass



Esses valores são atualizados automaticamente conforme os filtros selecionados.



Gráficos Disponíveis



Distribuição por Auto Renewal (Sim / Não)



Total por Tipo de Plano



Comparação entre serviços



Boas Práticas



Utilize os filtros para análises específicas



Combine filtros para análises mais detalhadas



Não altere estruturas, tabelas ou fórmulas



Utilize apenas as áreas destinadas à interação
